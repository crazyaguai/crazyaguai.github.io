<hexo>创建Hexo-github博客流程

简介

  • HEXO快速、简单且功能强大的 Node.js 博客框架。
    https://hexo.io/zh-cn/

安装

  1. 全局安装git以及nodeJs
  2. 全局安装hexo
  1. 新建项目文件夹
  1. 进入项目文件夹，初始化hexo
  1. 安装依赖
  1. 启动服务
  • 访问 http://127.0.0.1:4000/
  1. 此时可以在本地访问服务，下面进行提交代码到git的步骤，首先在 github (https://github.com/) 上创建项目，注意项目名字有格式要求必须为：[github用户名].github.io，否则不能使用，参考：https://github.com/crazyaguai/crazyaguai.github.io 名称。
  2. 安装hexo-deployer-git提交代码到git用，在项目中执行
  1. 配置根路径_config.yml文件
  1. 配置完成就可以将Hexo博客提交代码到github上了，执行
  1. 打开你的github hexo项目地址查看，例如 https://crazyaguai.github.io/

目录结构

  • _config.yml：全局配置文件
  • package.json：依赖
  • scaffolds：文章脚手架，根据文件夹下的配置构建文章
  • source：新建文章保存在这个文件夹下，在此修改文章
  • themes：主题目录

常用命令

  • hexo new “postName” #新建文章
  • hexo new page “pageName” #新建页面
  • hexo clean #清除缓存
  • hexo generate #生成静态页面至public目录
  • hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
  • hexo deploy #将.deploy目录部署到GitHub

命令简写

  • hexo n “我的博客” == hexo new “我的博客” #新建文章
  • hexo p == hexo publish
  • hexo g == hexo generate#生成
  • hexo s == hexo server #启动服务预览
  • hexo d == hexo deploy#部署
